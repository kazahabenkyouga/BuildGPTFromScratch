# NanoGPT

PyTorch を使って GPT を一から実装する学習プロジェクトです。

## 参考

このプロジェクトは、Andrej Karpathy 氏の動画 **「Let's build GPT: from scratch, in code, spelled out」** を参考にしています。

- 動画: [Let's build GPT: from scratch, in code, spelled out](https://www.youtube.com/watch?v=kCc8FmEb1nY&t=1436s)
- シリーズ: [Neural Networks: Zero To Hero](https://karpathy.ai/zero-to-hero.html)
- 参考コード（公式リポジトリ）: [karpathy/ng-video-lecture](https://github.com/karpathy/ng-video-lecture)

参考コードは第三者のリポジトリのため、このリポジトリには含めていません。ローカルで使う場合は clone してください。

## 環境

- **conda 環境:**
- **Python:** 3.11
- **PyTorch:** CUDA 対応版（RTX 5060 等）
